# LuminarGame — Кращий Minecraft сервер

**LuminarGame** — це український Minecraft-сервер з реалістичною економікою, правилами та дружньою спільнотою. Сервер працює на версії **Minecraft Java 1.20.1** та доступний для всіх гравців, як ліцензійних, так і на піратських копіях.

## Особливості сервера
- Безпечний спавн і дружня атмосфера  
- Захист територій та ресурсів  
- Чесна економіка та торгівля між гравцями  
- Зручні команди для комфортної гри  
- Регулярні івенти, розіграші та події  
- Прозора система донату без переваг у грі  

---

## Правила сервера
1. Не використовувати чіти, автоклікери, X-ray  
2. Не порушувати правила чесної гри та не експлуатувати баги  
3. Поважати інших гравців і дотримуватися етики спілкування  
4. Заборонена реклама без дозволу адміністрації  
5. Виконувати рішення адміністрації  

---

## Команди
- `/help` — отримати інформацію про команди  
- `/spawn` — повернутися на спавн  
- `/home` / `/sethome` — встановити та телепортуватися додому  
- `/warp` — телепортація до зон  
- `/msg` — приватні повідомлення
